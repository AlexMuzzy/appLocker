Setting up the appLocker

  • Enable permissions to use the app correctly.
  • Setup the password for your app.
  • Select which apps you want to lock.
  • close the app and watch your apps become locked from any threatening intruders!

Enabling the correct permissions

Firstly, once the app opens for the first time, be sure to enable the prompted permissions to allow you to properly use the app.

Setup a password

Now, you will need to set up a password by first tapping on the menu icon at the top right of the app. Then, select Settings, and tap on Reset password.

Selecting which apps to lock

Finally, select which apps you would like to password lock on your phone.

You're all set!

Any app you now decide to lock will now force you to authenticate the app!
